# email-administration-system
An OOP-based Java project implementing an Email Administration system with features for password generation, mailbox capacity management, and department-specific email formatting.

## 🚀 Features
- **Email Generation**: Automatically creates emails in the format `firstname.lastname@://company.com`.
- **Department Assignment**: Assigns employees to Sales, Development, or Accounting departments.
- **Random Password Generation**: Generates a secure, random 8-character password upon account creation.
- **Account Management**: Functionality to change passwords, set mailbox capacity, and define alternate email addresses.

## 🛠️ Technologies Used
- **Java**: Core programming language.
- **Object-Oriented Programming (OOP)**: Utilizes classes, constructors, and encapsulation.
